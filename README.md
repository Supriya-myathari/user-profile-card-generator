# 👤 User Profile Card Generator

A simple and responsive Flask web application that allows users to create a personalized profile card by entering their name, bio, and profile image URL. The application processes user input using POST requests and dynamically displays the profile information on the webpage.

## 🚀 Features

- 📝 Enter your name and bio
- 🖼️ Add a profile image using an image URL
- 🔄 Handles form submission using POST requests
- 🎨 Displays a styled profile card dynamically
- 📱 Responsive and clean user interface

## 🛠️ Technologies Used

- Python
- Flask
- HTML5
- CSS3
- Jinja2 Templates

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/your-username/user-profile-card-generator.git
```

### 2. Navigate to the project folder

```bash
cd user-profile-card-generator
```

### 3. Create a virtual environment (Optional)

```bash
python -m venv venv
```

### 4. Activate the virtual environment

**Windows**

```bash
venv\Scripts\activate
```

### 5. Install Flask

```bash
pip install flask
```

### 6. Run the application

```bash
python app.py
```

### 7. Open in your browser

```
http://127.0.0.1:5000
```
